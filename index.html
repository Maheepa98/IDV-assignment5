IDV Assignment 05 – Integrated Visualisations (A1–A4)

This single page combines my work from Assignment 01–04: an API-driven data table (A1), SVG transformations (A2), an interactive D3 chart with transitions (A3), and a complex/grouped data visualisation (A4). All four sections share a simple, consistent card layout, typography and spacing.

When integrating four separate assignments, the main challenges were avoiding CSS/ID conflicts and making sure each JavaScript visualisation runs correctly inside a shared HTML structure. The most difficult part was merging Assignment 3 and 4, because both rely on D3, specific SVG sizes and dynamic transitions. I solved this by: (1) giving each assignment its own container, (2) using separate initialisation functions called after DOMContentLoaded, and (3) keeping IDs and styles scoped to their own sections so that the charts do not interfere with each other.
